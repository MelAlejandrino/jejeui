jeje/ui is a collection of reusable components built on top of
[shadcn/ui](https://ui.shadcn.com).

Practical, copy-paste React components designed for dashboards, admin panels,
and internal tools — but flexible enough for any project.

All components follow shadcn/ui conventions — install them directly into your
project using the shadcn CLI. No extra dependencies, no wrappers, just code you own.
